Демонстрация: Pure vs Impure Pipes

Это новая, упрощенная демонстрация. Она показывает, что `pure` пайп реагирует только на смену ссылки на массив, а `impure` — на любые изменения.

Pure Pipe (`pure: true`)

Сработает только при полной замене массива. Мутацию проигнорирует.

Отображаемый список (видит сигнал):

    @for (item of pureItems() | pure; track item) {
  • {{ item }}
    • }

Состояние скрытого массива (мутируется напрямую):

{{ pureJson() }}

Impure Pipe (`pure: false`)

Сработает даже при мутации массива.

    @for (item of impureItems() | impure; track item) {
  • {{ item }}
    • }
