Структурная директива: *appUnless

Эта директива работает как *ngIf, но наоборот: отображает элемент, когда условие ложно.

Этот блок виден, потому что isBoxVisible равно false. Директива *appUnless сработала!

@if (isBoxVisible()) {

Сейчас блок для директивы *appUnless скрыт.

}

Атрибутивная директива: [appHighlight]

Эта директива изменяет стиль элемента при взаимодействии с ним.

Наведи на меня, чтобы увидеть атрибутивную директиву в действии.
